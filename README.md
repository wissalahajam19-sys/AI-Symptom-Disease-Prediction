# AI-Symptom-Disease-Prediction

This project predicts diseases from symptoms using Machine Learning.

## Dataset
Health symptom dataset from Kaggle

## Model
Random Forest Classifier

## Results
Accuracy: 92%

## Files
- AI_Symptom_Disease_Prediction.ipynb : training + evaluation
- disease_model.pkl : trained model
- vectorizer.pkl : text vectorizer
- label_encoder.pkl : labels encoder
